# nchu_cockpit_panel

#### 介绍
模拟仪表盘项目

#### 软件架构
软件架构说明
基于EGE图形库
通过XPC实现与Xplane11的UDP数据传输


#### 安装教程

1.  在配置文件中修改设置后打开exe文件

#### 使用说明

1.  在配置文件中设置IP地址
2.  打开Xplane11设置传输即可实现仪表盘的同步显示
